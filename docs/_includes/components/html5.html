HTML(5)

Block-level elements should not be nested within an inline element.

Examples

Sectioning Tags as first child within another sectioning tag

article as first child within article

Sectioning tag using improper role attribute

a section tag with the role="banner"

Use the <main> rather than an id or class

a tag with the id="main"

Use the <header> rather than an id or class

a tag with the id="header"

Use the <footer> rather than an id or class

a tag with the id="footer"
